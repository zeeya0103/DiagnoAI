# 🩺 DiagnoAI

An AI-powered healthcare assistant platform designed to simplify diagnostic services through intelligent automation and patient interaction.

🌐 Live Demo: https://diagno-ai-3bp8.vercel.app

---

## 🚀 Features

### 📄 Diagnostic Data Extraction (OCR)
- PDF report analysis interface
- Diagnostic data extraction module
- Ready for OCR integration
- Medical report processing workflow

### 💬 Context-Aware AI Chat
- Interactive healthcare chatbot interface
- User inquiry system
- AI-powered consultation module
- Context-aware response framework

### 🏠 Home Sample Collection Scheduler
- Schedule home diagnostic sample collection
- Patient-friendly appointment interface
- Healthcare service accessibility

### 🔐 Authentication System
- Secure login functionality
- User authentication support
- JWT-based backend architecture

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/zeeya0103/DiagnoAI.git
cd DiagnoAI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
uvicorn app.main:app --reload
```
